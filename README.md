## ✨ Key Features

### 🎓 Student Dashboard
- To-do list with wellbeing tasks
- Exercise tracker
- Timetable & exam scheduler
- Royalty points & rewards system
- SOS emergency button
- **NEW: Wellness Hub** 🌟

### 🌟 Wellness Hub (NEW!)
- **🎮 Interactive Games**
  - Memory Match game
  - Breathing exercises
  - Word puzzles
  
- **🎵 Music & Meditation**
  - Relaxation music player
  - 4-track playlist
  - Play/pause controls
  
- **🏆 Leaderboard**
  - Top 5 students ranking
  - Points competition
  - Gold/Silver/Bronze medals
  
- **📚 Digital Library**
  - 8 free storybooks
  - Educational content
  - Inspirational stories
  
- **😊 Mood Tracker**
  - Daily mood logging
  - Journal entries
  - Weekly patterns

### 🧠 AI-Powered Features (NEW!)
- **AI Risk Engine**
  - 4-category scoring (Mental, Lifestyle, Digital, Physical)
  - Overall Wellbeing Index (0-100)
  - Trend detection (Improving/Stable/Worsening)
  - Color-coded risk levels (Green/Yellow/Red)
  - "Why it changed" explanations

- **Persona Detection**
  - Balanced Student ✨
  - Screen-Overloaded 📱
  - Burnt-Out 😰
  - Sleep-Deprived 😴
  - Sedentary 🪑

- **Instant AI Feedback**
  - Real-time risk calculation on check-in
  - Comprehensive feedback modal
  - Personalized tips
  - Credits earned notification

### 💎 Gamification System
- **Wellbeing Credits**
  - Earn through healthy behaviors
  - Validation & consistency checks
  - Streak tracking
  - Rewards redemption

### 🚨 Alert System (Ethical & Tiered)
- **Tier 0:** Self-help resources
- **Tier 1:** Counselor notification
- **Tier 2:** Parent alert (consent-based)
- **Tier 3:** Emergency protocol
- All alerts show reason, consent status, recommended action

### 🔍 Transparency
- **How It Works Page**
  - AI explainability
  - Score calculation details
  - Privacy information
  - FAQs

### 👨‍👩‍👧 Parent Dashboard
- **Positive notifications** (not surveillance!)
- Student progress with consent
- Wellness trends

### 🧑‍⚕️ Counselor Dashboard
- Early warning alerts
- Student support tools
- Resource library

### 📊 Admin Dashboard
- Policy analytics
- Trend visualizations
- Risk pattern analysis
- Anonymous aggregated data

## 🧪 Testing Flow

1. **Login** → Use demo credentials
2. **Student Dashboard**:
   - Add a task
   - Complete tasks to earn points
   - Log exercise activities
   - Redeem rewards
   - Try SOS button
   - **Click "🌟 Wellness Hub"** button
3. **Wellness Hub**:
   - Play **Memory Match** game
   - Try **Breathing Exercise**
   - Play music with **Music Player**
   - Check your **Leaderboard** rank
   - Browse **Digital Library** books
   - Log your mood in **Mood Tracker**
4. **Switch Languages** → Select Hindi from dropdown
5. **Toggle Theme** → Click moon/sun icon
6. **Logout** → Login as different role (parent/counselor/admin)
